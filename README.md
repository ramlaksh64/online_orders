# Online Store Sales Dashboard - Excel Project

## Project Overview
This project is an Excel-based Online Store Sales Dashboard created as part of a Data Analytics learning project. The dataset was cleaned, processed, and analyzed to generate meaningful business insights through an interactive dashboard.

## Objectives
- Clean and preprocess raw sales data
- Remove duplicate and inconsistent records
- Analyze sales performance and customer trends
- Create an interactive dashboard using Excel

## Data Cleaning Process
The following steps were performed before analysis:
- Removed duplicate records
- Handled missing values
- Standardized data formats
- Corrected inconsistent entries
- Organized columns for analysis

## Tools & Features Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- KPI Cards
- Data Cleaning Techniques

## Dashboard Insights
The dashboard helps analyze:
- Total Sales
- Order Trends
- Top Performing Products
- Customer Purchase Behavior
- Region-wise Sales Performance
- Monthly Revenue Analysis

- Data Cleaning
- Data Analysis
- Dashboard Design
- Business Insight Generation
- Excel Visualization

Successfully developed an interactive Excel dashboard that transforms raw online store data into actionable business insights.
